# ScareBandB Docusaurus App

This folder contains the Docusaurus app that renders the repo-root `../Docs` tree.

## Local Development

```bash
npm install
npm start
```

## Production Build

```bash
npm run build
npm run serve
```

## Important

- Author docs in `../Docs`, not in `website/docs`.
- `sidebars.ts` is the thin autogenerated shell; normal doc navigation comes from `Docs/` plus `_category_.json` and `sidebar_position`.
- Deployment metadata lives in `docusaurus.config.ts`.
- Use `docs-tools start` and `docs-tools stop` from the repo root for the normal local dev server flow.
- Use `docs-tools status` to inspect the tracked dev server and `docs-tools doctor` to check local docs prerequisites.
- Use `docs-tools` from the repo root for normal docs scaffolding, validation, and Docusaurus script passthrough.
