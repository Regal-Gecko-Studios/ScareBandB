# ScareBandB Docusaurus App

This folder contains the Docusaurus app that renders the repo-root [Docs](/C:/Users/Rim28/Projects/ScareBandB/Docs) tree.

## Local Development

```bash
npm install
npm start
```

## Production Build

```bash
npm run build
npm run serve
```

## Important

- Author docs in `../Docs`, not in `website/docs`.
- Navigation is defined in `sidebars.ts`.
- Deployment metadata lives in `docusaurus.config.ts`.
