# Core Session Goal
The goal of a session is for the ghost players to make the NPC guests leave the Air B&B before the end of the final round.

- Current placeholder structure is 3 rounds across a weekend:
    - Friday
    - Saturday
    - Sunday
- Each round is split into:
    - Day
    - Night
    - Twilight
- The primary win condition is pushing the NPC group into a fear threshold where the majority hit critical fear and decide to leave.
- If that happens, the whole group leaves and the players win the session.
- If the players fail to force the NPCs out by the final round, the session ends in failure.
